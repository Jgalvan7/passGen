Password Gen

Generador de password aleatorios

Bienvenido a este pequeño proyecto, el generador de password aleatorios. Por sí solo no es gran cosa lo sé, pero esta pequeña porción de código la puedes implementar en muchas otros sistemas, para autentificar usuario o verificar cuentas por ejemplo, sé que este tipo de módulos ya vienen dados en otras librerías o sistemas, pero me gusta programar este tipo de cosas básicas, poder crear yo mismo el código me ayuda a mejorar mis habiliades y alimenta mi imaginación, empezar por cosas simples como esta y quien sabe donde se puede llegar.

Diseñé dos versiones, una que genera el password cada vez que pulsas el botón y otra que simula los códigos OTP para la típica autentificación en dos pasos.

Puedes probar este proyecto aquí mismo para ver si te gusta o puedes descargar el código desde el repositorio e implementarlo donde desees.
